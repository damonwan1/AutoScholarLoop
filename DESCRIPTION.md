# AutoScholarLoop Description

Short description for GitHub:

> A multi-agent AUTO Research loop for idea discovery, experiment execution,
> evidence-grounded paper writing, quality audit, and live Web observation.

Long description:

> AutoScholarLoop is an open-source research automation framework that simulates
> a small research group. It builds a field evidence archive, runs professor
> decision loops, executes PhD-style experiment loops, writes papers from
> claim-evidence tables, and applies quality gates for novelty, citation,
> reproducibility, and unsupported claims. It includes both a Python CLI and a
> Vue Web console for launching and observing research loops.
